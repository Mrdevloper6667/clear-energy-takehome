## Why npm workspaces?
We chose `npm workspaces` over Turborepo or Nx to keep things simple and adhere strictly to the fundamental requirements of the assignment without introducing unnecessary overhead or complex build systems right away. The goal is a clean, minimal monorepo that supports Expo and a shared package quickly, which npm workspaces provides natively.
